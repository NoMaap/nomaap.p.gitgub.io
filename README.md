Privacy Policy

Last Updated: March 6, 2026

1. Data Collection
No Maap does not collect, store, or share personal data.
Location data is used only locally on your device to provide navigation functionality.
This data is not transmitted to No Maap servers.
2. Location Services
Location access is required for the core functionality of the app.
All location processing is handled using
Apple's Core Location and MapKit frameworks. No Maap does not independently track or retain location history.
3. Third-Party Frameworks
No Maap depends on Apple-provided frameworks for mapping and positioning services.
Apple may process data in accordance with its own privacy policies.
